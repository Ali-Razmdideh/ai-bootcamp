Part 1 · 0:00 – 0:30

Scaling & pipelines

StandardScaler, log1p

Why scale at all?

  • Plain LR doesn't care — but regularized LR does (Ridge/Lasso in Course 6).
  • Distance-based methods (KNN, SVM next week) crumble without scaling.
  • Neural nets converge faster on standardized inputs.

Pipeline = scaler + estimator, one object

from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LinearRegression

pipe = Pipeline([
    ('scale', StandardScaler()),
    ('lr', LinearRegression()),
])
pipe.fit(X_train, y_train)
pipe.score(X_test, y_test)

Pipelines survive train/test splits — they fit the scaler on train only.

Skew? log1p it.

Many natural features (income, counts, prices) are right-skewed.

log1p(x) = log(1 + x)

log1p handles zeros (unlike plain log). Always sanity-check with a histogram.

Part 2 · 0:30 – 1:00

Polynomial features

underfit ↔ overfit

Polynomial features turn LR into a curve-fitter

from sklearn.preprocessing import PolynomialFeatures
PolynomialFeatures(degree=3).fit_transform(X)
# [x] -> [1, x, x^2, x^3]

Linear in the parameters β, non-linear in x. Still linear regression!

Bias / variance, visualized

  • d = 1 — too straight (underfit, high bias).
  • d = 2 — captures the curve (just right).
  • d = 15 — wagging tails (overfit, high variance).

How do you pick d without eyeballing? Cross-validation, next course.

Part 3 · 1:00 – 1:30

Encoding & interactions

ColumnTransformer is your friend

Different transforms for different columns

ColumnTransformer([
    ('num', StandardScaler(), numeric_cols),
    ('cat', OneHotEncoder(drop='first'), categorical_cols),
])

Interactions in bulk

PolynomialFeatures(interaction_only=True, include_bias=False)
# adds every x_i * x_j pair

Combinatorial explosion ahead — combine with Lasso (Course 6) to keep only the useful interactions.

Recap

Three things to leave with

  1. Scale numeric features before regularizing or measuring distances.
  2. Polynomial features turn LR into a flexible curve fitter. Pick degree with care.
  3. Pipelines + ColumnTransformer keep preprocessing honest under train/test splits.

Next: how do you pick the degree without fooling yourself? Cross-validation.
