SciPy stats

  


  

    
The SciPy stack at a glance

    
    
SciPy is a federation of subpackages — all built on NumPy arrays, all sharing one mental model.

  


  

    
A gallery of distributions

    
    
Same SciPy API for all four: .pdf/.pmf, .cdf, .ppf, .rvs, .fit.

  


  

    
Pick the right hypothesis test

    
    
Pick by question and data shape, not by habit. Every test below lives in scipy.stats.

Curve fitting — model, fit, residuals

    
    
scipy.optimize.curve_fit finds θ that minimises Σ(yᵢ − f(xᵢ; θ))². Always look at the residuals — a good fit leaves only noise.

    Part 3 · 1:00 – 1:30
    
Optimize + Grand Finale

  


  

    
The optimization landscape

    
    
A non-convex loss has many valleys. scipy.optimize.minimize follows the local slope — initial point matters.

  


  

    
minimize — the engine under every model

    
The Rosenbrock function is the classical test bed: a narrow curved valley with the minimum at (1, 1). Every supervised model is "define a loss, call an optimizer" — res.x is the argmin, res.fun the minimum value, res.nit the iteration count.
