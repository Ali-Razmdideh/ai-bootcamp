Matplotlib & Seaborn

make the data visible

Your first plot

import matplotlib.pyplot as plt

plt.plot([-3, -2, 5, 0], [1, 6, 4, 3])
plt.show()

Two arrays: x-coords and y-coords. plot() connects them; show() renders.

Plot a function

import numpy as np

x = np.linspace(-2, 2, 500)
y = x**2
plt.plot(x, y)
plt.show()
np.linspace + a vectorized expression + plt.plot = every function plot you'll ever make.

Labels, title, grid, axis limits

x = np.linspace(-2, 2, 500)
y = x**2

plt.plot(x, y)
plt.title("Square function")
plt.xlabel("x")
plt.ylabel("y = x**2")
plt.axis([-2.5, 2.5, 0, 5])      # [xmin, xmax, ymin, ymax]
plt.grid(True)
plt.show()
A plot without axis labels is decoration, not information. Label everything you ship.

Multiple curves on one axes

plt.plot([0, 100, 100,   0,   0], [0,   0, 100, 100,   0], "r-")
plt.plot([0, 100,  50,   0, 100], [0, 100, 130, 100,   0], "g--")
plt.axis([-10, 110, -10, 140])
plt.show()

Style strings: "r-" red solid, "g--" green dashed, "bo" blue dots, "k:" black dotted.

Subplots — a 2×2 grid

x = np.linspace(-1.4, 1.4, 30)

plt.subplot(2, 2, 1);  plt.plot(x, x);     plt.title("x")
plt.subplot(2, 2, 2);  plt.plot(x, x**2);  plt.title("x²")
plt.subplot(2, 2, 3);  plt.plot(x, x**3);  plt.title("x³")
plt.subplot(2, 2, 4);  plt.plot(x, x**4);  plt.title("x⁴")
plt.show()
subplot(rows, cols, idx) — index starts at 1 (top-left), counts left-to-right then top-to-bottom.

Saving

x = np.linspace(-1.4, 1.4, 30)
plt.plot(x, x**2)

savefig writes whatever is in the current figure. Call it before plt.show() if you're in a script.

3D surface — same vocabulary, one extra dimension

import matplotlib

x = np.linspace(-5, 5, 50)
y = np.linspace(-5, 5, 50)
X, Y = np.meshgrid(x, y)
R = np.sqrt(X**2 + Y**2)
Z = np.sin(R)

ax = plt.subplot(111, projection="3d")
ax.plot_surface(X, Y, Z, rstride=1, cstride=1,
                cmap=matplotlib.cm.coolwarm)
plt.show()

np.meshgrid turns 1-D x and y into 2-D coordinate grids — the lingua franca for any 2-D function.

Contour — the 2-D loss landscape

plt.contourf(X, Y, Z, cmap=matplotlib.cm.coolwarm)
plt.colorbar()
plt.show()
You'll see contour plots constantly when visualizing 2-parameter loss surfaces in ML.

Part 2 · 0:45 – 1:15

SciPy

probability and optimization

scipy.stats — every distribution speaks the same dialect

from scipy import stats

# Continuous
stats.norm.pdf(x, loc=0, scale=1)              # PDF
stats.norm.cdf(x)                              # cumulative
stats.norm.rvs(size=1000)                      # samples
stats.norm.fit(data)                           # MLE parameters

# Discrete
stats.binom.pmf(k=5, n=10, p=0.5)

# Hypothesis tests
stats.ttest_ind(group_a, group_b)
stats.pearsonr(x, y)

Every distribution: .pdf · .cdf · .rvs · .fit. Learn the verbs once, get 80+ distributions free.

Plot three Gaussians, on one axes

x = np.linspace(-4, 4, 200)
for mu, sigma in [(0, 1), (0, 0.5), (1, 1)]:
    plt.plot(x, stats.norm.pdf(x, mu, sigma), label=f"N({mu}, {sigma})")
plt.legend()
plt.title("Normal PDFs")
plt.show()

Fit a Gaussian to data

rng = np.random.default_rng(1)
samples = rng.normal(loc=2.0, scale=1.5, size=1000)

mu_hat, sigma_hat = stats.norm.fit(samples)    # MLE → ≈ (2.0, 1.5)

xs = np.linspace(samples.min(), samples.max(), 200)
plt.hist(samples, bins=30, density=True, alpha=0.5)
plt.plot(xs, stats.norm.pdf(xs, mu_hat, sigma_hat), lw=2, label="fitted")
plt.legend()

This is what a one-parameter "model" does — fit, then plot fit vs. data. Scale it up to millions of parameters and you have deep learning.

A 1-D loss landscape

The optimizer takes downhill steps until the gradient is ~0. This is the whole story of supervised learning.

scipy.optimize — what models actually do

from scipy import optimize

f = lambda x: (x - 3)**2 + np.sin(5 * x)
res = optimize.minimize(f, x0=0.0)

res.x       # ≈ argmin
res.fun     # min value
res.nit     # iterations
Every supervised model: define a loss L(parameters), then ask an optimizer to minimize it. fit in sklearn, backward() in PyTorch — all the same idea.

Local minima — why initialization matters

optimize.minimize(f, x0=0.0).x      # ≈ 2.95
optimize.minimize(f, x0=10.0).x     # ≈ 6.21   different local min!
Bumpy loss surfaces are everywhere in neural nets. This is why "different random seed → different model" is a real conversation, not folklore.

For multi-modal landscapes try global solvers: differential_evolution, basinhopping, dual_annealing.

The optimization API at a glance
