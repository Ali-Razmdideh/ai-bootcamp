0:00 – 0:45

Matplotlib: the anatomy of a plot

fig, ax = plt.subplots(figsize=(6, 4))
ax.plot(x, y, label='train')
ax.set_xlabel('epoch'); ax.set_ylabel('loss')
ax.set_title('Loss curve'); ax.legend()

Always use the object-oriented API (fig, ax). Avoids surprises when you have multiple subplots.

The two charts you'll draw most

  • Line plot — loss curves, time series.
  • Scatter — clusters, predictions vs. truth.
ax.plot(epochs, train_loss, label='train')
ax.plot(epochs, val_loss,  label='val')

ax.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')

Seaborn: stats plots in one line

sns.heatmap(df.corr(), annot=True, cmap='coolwarm')
sns.kdeplot(data=df, x='bill_length_mm', hue='species', fill=True)
sns.pairplot(df, hue='species')

Heatmap of correlations is the fastest way to spot redundant features. Use it before any modeling.

The optimizer demo — what models actually do

from scipy import optimize

f = lambda x: (x - 3)**2 + np.sin(5 * x)
res = optimize.minimize(f, x0=0.0)
print(res.x)   # ≈ argmin

Every supervised model: define a loss L(parameters), then ask an optimizer to minimize it. That's it.

Local minima — why initialization matters

Try x0 = 0 vs x0 = 10 on the same bumpy function and you'll get different answers. This is the conceptual seed of every "different random seed -> different model" conversation later.
