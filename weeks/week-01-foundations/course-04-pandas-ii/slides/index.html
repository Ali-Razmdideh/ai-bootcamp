Part 1 · 0:00 – 0:30

Missing data & column ops

Find the holes

titanic.isna().sum()       # NaNs per column

Always start cleaning by counting. isna().sum() is your microscope.

Three responses to a NaN

  • Drop — dropna(subset=['embarked']). Use when missingness is rare or non-informative.
  • Fill — fillna(df['age'].median()). Use when you can't afford to lose rows.
  • Interpolate — interpolate(axis=1). Use for ordered data with smooth structure.

Column ops — assign and query

(tips
   .assign(tip_pct=lambda df: df['tip'] / df['total_bill'])
   .query('tip_pct > 0.2 and time == "Dinner"'))

assign adds a derived column without mutating; query filters with a string expression. Both chain.

Part 2 · 0:30 – 1:00

Group, pivot, join

GroupBy — the workhorse

tips.groupby('day')['tip'].mean()
tips.groupby(['day', 'sex'])['tip'].agg(['mean', 'count'])

Split → apply → combine. Pass a single column or a list; pass a single agg or a list of aggs.

Pivot — reshape from long to wide

pd.pivot_table(
    more_grades,
    index='name', columns='month', values='grade',
    margins=True)

Merge — SQL joins for DataFrames

pd.merge(city_loc, city_pop, on='city')             # INNER
pd.merge(city_loc, city_pop, on='city', how='outer') # OUTER
pd.merge(city_loc, city_pop, on='city', how='left')  # LEFT

Same vocabulary as SQL. how= picks the join type.

Concat — stack rows or columns

pd.concat([a, b], ignore_index=True)   # vertical
pd.concat([a, b], axis=1)              # horizontal
Part 3 · 1:00 – 1:30

Time series

A DatetimeIndex is the whole game

dates = pd.date_range('2016-10-29 17:30', periods=12, freq='h')
ts = pd.Series(temps, index=dates)

Once the index is datetimes, every time-aware method unlocks: .dt, .resample, partial-string slicing.

Resample — change the frequency

ts.resample('2h').mean()             # downsample
ts.resample('15min').interpolate()   # upsample

Downsampling aggregates; upsampling interpolates. Pick the verb that matches your direction.

Real CSV — stock-prices

sp = load_dataset('stock-prices')
sp.columns                       # ['AAPL_x', 'AAPL_y']
sp['AAPL_x'] = pd.to_datetime(sp['AAPL_x'])
sp = sp.sort_values('AAPL_x').set_index('AAPL_x')

sp['AAPL_y'].resample('W').mean().plot()

Load, parse, sort, set the index, resample, plot. Five lines.

Recap

Three things to leave with

  1. Clean by counting first. isna().sum() before any drop/fill decision.
  2. GroupBy + pivot + merge cover most reshape work. Three verbs, most jobs.
  3. Make your index a DatetimeIndex and resample becomes a one-liner.
