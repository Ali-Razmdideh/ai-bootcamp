Part 1 · 0:00 – 0:30

The Figure / Axes model

Three nouns

  • Figure — the page.
  • Axes — one rectangular plot area on the page.
  • Artist — anything you draw: line, marker, text, legend.

Every method follows from these three. fig.savefig, ax.plot, ax.legend — the prefix tells you who owns the action.

Anatomy of a figure

Every element on screen has a name — and a method on either fig or ax.

Title, ticks, spines, gridlines, legend — each one is an ax.set_* or ax.* call.

Figure vs Axes vs Axis

Three nested containers — and yes, "Axes" is not the plural of "Axis."

Figure ⊃ Axes ⊃ Axis. "Axes" means one plot region (it has both an x and y Axis); "Axis" means a single number line.

Pyplot vs object-oriented API

Two ways to call the same library. Prefer OO for anything multi-axes.

pyplot (quick)

import matplotlib.pyplot as plt

plt.plot(x, y)
plt.title('Quick')
plt.xlabel('x')
plt.show()

Implicit "current figure." Fine for one-off plots.

Object-oriented (controllable)

import matplotlib.pyplot as plt

fig, ax = plt.subplots()
ax.plot(x, y)
ax.set_title('Controllable')
ax.set_xlabel('x')
plt.show()

Explicit handles. Scales to grids, insets, callbacks.

Prefer OO for anything multi-axes — the prefix (fig. vs ax.) tells you who owns the action.

Build the anatomy yourself

Recreate the anatomy figure live: sin(x), title, xlabel, ylabel, legend, grid. Every label is an ax.set_* call.

Colormap reference

    
A colormap turns a scalar into a color. Pick perceptually uniform maps for ordered data.

    
    
viridis is perceptually uniform — equal data steps look like equal color steps. Use coolwarm for diverging data centered on zero.

  


  

    
What is a bin?

    
A bin is a fixed-width interval on the x-axis. Each bar's height is the count of values that fell into that interval.

    
    
    
      
  • bins=30 — number of equal-width bins (an int).
    • 
      
  • bins=[0, 1, 2, 5, 10] — explicit edges (a sequence). Unequal widths are allowed.
    • 
      
  • bins='auto' / 'sturges' / 'fd' — let numpy pick a rule.
    • 
    

    
Too few bins hides the shape; too many turns it into noise. Try a few values before trusting what you see.

Subplots & saving

  


  

    
Subplots grid — shape of the axes array

    
One call returns a Figure and a 2-D NumPy array of Axes. Index each cell by axes[row, col].

    
    
fig, axes = plt.subplots(2, 3) — axes.shape == (2, 3). Use axes.flat to iterate row-major.

  


  

    
A 2×2 grid in one call

    
plt.subplots(2, 2) returns a Figure and a 2-D NumPy array of Axes. sharex=True aligns the x-axes; fig.tight_layout() stops titles from colliding.

    Part 4 · 1:30 – 2:00
    
Beyond the basics

    
Ten everyday tools the official guide hammers — and you will hit within a week of real work.

  


  

    
Annotations & text

    
ax.text drops a string at (x, y). ax.annotate adds an arrow from text to a target point — perfect for calling out a peak, an outlier, or a regime change.


fig, ax = plt.subplots()
x = np.linspace(0, 4*np.pi, 200)
ax.plot(x, np.sin(x))
peak = (np.pi/2, 1.0)
ax.annotate('first peak', xy=peak, xytext=(2.5, 1.3),
            arrowprops=dict(arrowstyle='->', color='#b45309'))
ax.text(4*np.pi - 0.5, -0.9, 'end of window', ha='right', color='#64748b')

    
xy is the arrow tip; xytext is where the label sits.

  


  

    
Axis scales — log, symlog, logit

    
One call rescales an axis. Use 'log' for orders-of-magnitude data (loss curves, populations), 'symlog' when the data crosses zero, 'logit' for probabilities near 0 or 1.


epochs = np.arange(1, 101)
loss = 1.0 / epochs + 0.01

fig, ax = plt.subplots()
ax.plot(epochs, loss)
ax.set_yscale('log')          # linear in log-space
ax.set_xlabel('epoch'); ax.set_ylabel('loss')

    
A straight line on a log-y plot means exponential decay. Always check both scales before claiming "it plateaued."

  


  

    
Date & time ticks

    
Matplotlib understands datetime and numpy.datetime64 on either axis. matplotlib.dates gives you locators (where ticks go) and formatters (how they render).


import matplotlib.dates as mdates
dates = pd.date_range('2026-01-01', periods=120, freq='D')
y = np.cumsum(np.random.default_rng(0).standard_normal(120))

fig, ax = plt.subplots()
ax.plot(dates, y)
ax.xaxis.set_major_locator(mdates.MonthLocator())
ax.xaxis.set_major_formatter(mdates.DateFormatter('%b %Y'))
fig.autofmt_xdate()   # rotate labels so they fit

  


  

    
Colorbars

    
A colorbar turns the colormap legend into something readable. The trick: scatter, imshow, and friends return a mappable; pass it to fig.colorbar together with the Axes it should attach to.


rng = np.random.default_rng(0)
x, y, z = rng.standard_normal((3, 300))

fig, ax = plt.subplots()
sc = ax.scatter(x, y, c=z, cmap='viridis', s=20)
cbar = fig.colorbar(sc, ax=ax)
cbar.set_label('z value')

    
Without ax=ax the bar steals space from whatever Axes is current — surprising in subplot grids.

  


  

    
imshow — images & heatmaps

    
Any 2-D array becomes a heatmap. Confusion matrices, attention maps, raw pixels — same call.


rng = np.random.default_rng(0)
M = rng.random((8, 8))

fig, ax = plt.subplots()
im = ax.imshow(M, cmap='magma', origin='lower',
               extent=[0, 8, 0, 8], aspect='equal')
fig.colorbar(im, ax=ax)

    
    
      
  • origin='lower' — put row 0 at the bottom (math convention). Default is top (image convention).
    • 
      
  • extent=[x0,x1,y0,y1] — map array indices to data coordinates.
    • 
      
  • aspect='auto' — let cells stretch to fill; 'equal' keeps them square.
    • 
    

  


  

    
Constrained layout — the modern tight_layout

    
tight_layout() runs once at call time and can leave you to tweak it again whenever you change a label. layout='constrained' runs every draw and adapts automatically — including for colorbars and suptitles.


fig, axes = plt.subplots(2, 2, figsize=(8, 5),
                         layout='constrained')
for ax in axes.flat:
    ax.plot([0, 1], [0, 1]); ax.set_title('a long title')
fig.suptitle('layout="constrained" handles this automatically')

    
Prefer layout='constrained' for any new plot with multiple subplots, colorbars, or a suptitle.

  


  

    
subplot_mosaic — layouts you can read

    
Asymmetric layouts used to require gridspec. subplot_mosaic takes an ASCII map and gives you back a dict keyed by the labels you wrote.


fig, axd = plt.subplot_mosaic(
    '''
    AAB
    CCB
    ''', figsize=(8, 4), layout='constrained')

axd['A'].plot([0,1,2], [0,1,4]); axd['A'].set_title('A — wide top')
axd['B'].hist(np.random.default_rng(0).standard_normal(200))
axd['C'].plot([0,1,2], [2,1,0]); axd['C'].set_title('C — wide bottom')
axd['B'].set_title('B — full-height right')

  


  

    
Twin axes — two y-scales, one chart

    
ax.twinx() returns a second Axes that shares the x-axis. Use it sparingly — readers can be misled by the independent y-scales — but it is the right call for comparing series in different units.


x = np.linspace(0, 10, 200)
fig, ax = plt.subplots()
l1, = ax.plot(x, np.sin(x), color='#2563eb', label='signal')
ax.set_ylabel('signal', color='#2563eb')

ax2 = ax.twinx()
l2, = ax2.plot(x, np.exp(x/5), color='#b45309', label='magnitude')
ax2.set_ylabel('magnitude', color='#b45309')

ax.legend(handles=[l1, l2], loc='upper left')

  


  

    
rcParams — defaults you set once

    
Every default — figure size, DPI, font, line width, grid — lives in plt.rcParams. Set it at the top of a notebook and the rest of the session inherits.


import matplotlib as mpl
mpl.rcParams['figure.dpi']    = 120
mpl.rcParams['figure.figsize'] = (7, 4)
mpl.rcParams['axes.grid']      = True
mpl.rcParams['lines.linewidth'] = 1.8

print(plt.style.available[:5])   # built-in style sheets
# plt.style.use('seaborn-v0_8-whitegrid')   # apply one globally

    
The same keys live in your ~/.matplotlibrc file for persistent defaults across projects.

  


  

    
Legend placement

    
loc= picks one of nine corners; bbox_to_anchor places the legend at an arbitrary figure coordinate. Need two legends on one Axes? Add the first manually as an artist, then call ax.legend again.


fig, ax = plt.subplots(figsize=(7, 4), layout='constrained')
l1, = ax.plot([0,1,2], [0,1,4], label='line A')
l2, = ax.plot([0,1,2], [4,2,1], label='line B')
l3, = ax.plot([0,1,2], [1,2,3], '--', label='reference')

leg1 = ax.legend(handles=[l1, l2], loc='upper left',
                 title='series')
ax.add_artist(leg1)               # keep leg1 when we call legend again
ax.legend(handles=[l3], loc='center left',
          bbox_to_anchor=(1.02, 0.5), title='lines')

    
bbox_to_anchor=(1.02, 0.5) = just outside the right edge, vertically centered. Pair with layout='constrained' so the figure expands to fit.
